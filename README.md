# OS Process Scheduling Practice Tool

**Built by Mawo** | For UMBB CS Students (ENG2)

## What is this?

An interactive drag-and-drop tool to practice CPU scheduling algorithms for Operating Systems exams. No more drawing Gantt charts on paper!

## Features

✅ Configurable CPUs, I/O devices, and timeline length  
✅ Drag-and-drop process blocks (P1-P5) onto the timeline  
✅ Adjustable Ready Queue and I/O Queue rows  
✅ Click to remove, drag to move blocks  
✅ Reset button to start over  
✅ Matches UMBB TD exercise format exactly  

## How to Use

1. **Configure** your system (CPUs, I/O devices, timeline length)
2. Click **Apply** to generate the grid
3. **Drag** process blocks (P1, P2, P3, etc.) onto the timeline
4. **Practice** solving scheduling exercises from your TD
5. **Compare** with the official solutions
6. **Reset** and try again!

## Algorithms Supported

Practice these scheduling algorithms:
- FCFS (First Come First Served)
- SJF (Shortest Job First)
- SRTF (Shortest Remaining Time First)
- Priority Scheduling
- Round Robin (RR)
